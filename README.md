# Gooodcase

一个基于Next.js的现代化Web应用。

## 功能特点

- 使用Next.js 14构建
- 使用Tailwind CSS进行样式设计
- 支持拖拽功能
- 响应式设计
- 现代化UI组件

## 技术栈

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Radix UI
- DND Kit

## 开始使用

1. 克隆仓库
```bash
git clone [repository-url]
```

2. 安装依赖
```bash
npm install
```

3. 运行开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 `http://localhost:3000`

## 构建部署

```bash
npm run build
npm start
```
